JavaScript + CSS でプレゼンする

なんで JavaScript + CSS なの？

  • どこでもテキストエディタだけで編集できる
  • ブラウザで見ることができるので
    ファイルを持ち歩く必要がない
  • 共有が楽ちん

なら s6 でいいじゃんという話ですが…

iPhoneOS や Android など
Webkit ベースのブラウザを搭載したモバイル端末で
(たぶん)高速に動作します！

アニメーションの設定

  1. CSS でアニメーションを指定

    /* opacity の変化は線形に1秒かけておこなう */
    transition-property: opacity 1s linear;

  2. あとは JavaScript でスタイルの値を変更するだけ！

    // id が 'picture' の要素が1秒かけて透明になります
    $("#picture").css("opacity", "0")

  3. 動作には任意の JavaScript の関数が設定できます

    // 見直しのために経過時間を記録
    recorder.push(new Date())

HTML なのでいろいろできます

  • リンクとか
  • 外部ページを表示してみたり
  • <video>タグを使ってみたり

MobileSafari だと崩れるみたいですが…

制限など

  • クロスブラウザではありません
    Safari や Chrome など、Webkit 系のブラウザでのみ動作確認
  • アニメーションの指定方法が煩雑
    複雑な動きをさせる場合には CSS を自分で書きます
  • ドキュメントは特にありません
    興味があったらソース読んでね！汚いけど :)
